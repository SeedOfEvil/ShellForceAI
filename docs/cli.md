## Interactive CLI

Running `shellforgeai` with no subcommand starts an interactive ShellForgeAI session with banner, workspace trust prompt, slash commands, and natural-language routing.

Supported slash commands: `/help`, `/exit`, `/quit`, `/doctor`, `/model`, `/tools`, `/audit`, `/workspace`, `/mode`, `/profile`, `/clear`, `/raw on|off`, `/context minimal|standard|full`, `/diagnose`, `/ask`, `/research`, `/plan`.
